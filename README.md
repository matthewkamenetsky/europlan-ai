Europlanner is an AI-powered Schengen travel planner that generates and critiques itineraries using a planner–critic LLM pipeline grounded in real geographic data. A user enters a starting city, trip length, and interests. The planner LLM generates a day-by-day itinerary. The critic LLM then evaluates it for realism, flagging issues like excessive travel distances or unrealistic pacing, and scores it. A replan endpoint then takes the critique and generates an improved itinerary.
